详细描述

获取几个参数中的最大值或某类型能表示的最大值. 用法: max.able<T1,T2...> -> bool T...等类型能否获得参数最大值. max.get_limit_able<T> -> bool 能否获得T类型能表示的最大值. max(v1,v2...) -> auto 获取v...的最大值. max(type_info<T>) -> auto 获取T能表示的最大值.
